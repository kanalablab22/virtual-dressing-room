# 👗 Virtual Dressing Room（バーチャル試着室）

モデルの服をAIで着せ替えたいとき、「どんな服にしよう」「ポン出しだと似た服ばかり」「バッグと合わない」……そんな悩みを解決するツールです。

モデルと服を選ぶだけで、ChatGPTに貼り付けるプロンプトが自動で完成します。あとはそれを貼るだけで、選んだ服に着替えたモデルの画像がつくれます。

公開URL 👉 https://kanalablab22.github.io/virtual-dressing-room/

## これは何ができるツール？

- たくさんの服の中から、着せたいコーデを選べます
- 選んだモデルに、選んだ服を着せる「ChatGPT用プロンプト」を自動で作ってくれます
- そのプロンプトをChatGPTに貼るだけで、着せ替え画像が完成します

服のカテゴリは「モデル・コーデ全身・夏服・キレイめ・トップス・ボトムス・ワンピース」など。今後も随時追加していきます。

---

## 必要な環境

- **OS**: macOS または Windows（どちらでもOK）
- **ブラウザ**: Chrome、Edge、Safari など（最近のものならOK）
- **ChatGPT**: 画像生成が使えるアカウント（着せ替え画像を作るときに使います）

インストールは不要です。上の公開URLを開くだけで使えます。

## 基本の使い方（3ステップ）

### ステップ0：ツールを開く

公開URL（https://kanalablab22.github.io/virtual-dressing-room/）をブラウザで開きます。
毎回同じパソコン・同じURLで開くのがおすすめです（追加した服がそのまま残ります）。

### ステップ1：モデルを選ぶ 🧍

左上の金色の「モデル」タブを押して、使いたいモデルをクリックします。
モデルはバッグ違い・ポーズ違いで複数用意されているので、作りたい雰囲気に近いものを選んでください。

### ステップ2：着せたいコーデ（服）を選ぶ 👗

カテゴリのタブ（コーデ全身・ワンピース・トップスなど）を切り替えながら、着せたい服をクリックします。
選んだ服は右側の「今日のコーデ」に入っていきます。

### ステップ3：プロンプトを作って、ChatGPTに貼る 📋

1. 右下の「プロンプトを生成」ボタンを押す
2. 「JSONをコピー」を押す
3. 表示された「添付順」のとおりに、画像をChatGPTに添付する
   - 「選択画像を番号付きで書き出す」ボタンを使うと、`01_モデル.jpg` `02_○○.jpg` のように添付する順番どおりのファイル名で画像がダウンロードされるので便利です
4. コピーしたプロンプトをChatGPTに貼り付けて送信

→ 選んだ服に着替えたモデルの完成です！🎉

## 服の追加のしかた

気になる服（ピンタレストやGoogle画像など）を、自分で追加できます。

### 方法1：コピー＆貼り付け（おすすめ）

1. 追加したい画像を右クリックして「画像をコピー」
2. ツールの画面上で貼り付け
   - **macOS**: `Command（⌘） + V`
   - **Windows**: `Ctrl + V`

### 方法2：ファイルをドラッグ＆ドロップ

保存済みの画像ファイルを、ツールの「服・モデル画像を追加」の枠にドラッグ＆ドロップします。
「ファイルを選ぶ」ボタンから選んでもOKです。

### コツ

- 貼り付ける前に、上の「追加先カテゴリ」で入れたい分類（トップス・ボトムスなど）を選んでおくと整理されます
- 服のカードの名前部分をクリックすると、「白ブラウス」などわかりやすい名前に変えられます（プロンプトにもその名前で出ます）

## 便利な機能

- **拡大して見る 🔍**：服のカードにマウスを乗せると出る拡大ボタンを押すと、大きなプレビューでじっくり見られます（画面の暗い部分をクリック、または Esc キーで閉じます）
- **単品／全身コーデの切り替え**：「今日のコーデ」に入れた服は、「単品」（その服だけ着せる）と「全身コーデ」（写真のコーデ一式を着せる）を切り替えられます
- **モデルの扱い**：「モデル画像を使う」のほか、「文章で指定」（年代や雰囲気を選ぶ）、「服だけ（物撮り）」も選べます
- **撮り方の指定**：全身・バストアップなどの構図、背景、画像の縦横比を選べます
- **リセット**：画面上部のタイトル「Virtual Dressing Room」をクリックすると、選んだ服・モデルがぜんぶリセットされます（登録した服自体は消えません）

## 【管理者向け】服を全員に公開する

自分で追加した服を、URLを見る全員にも表示させたいときの手順です。
（ふだん使うだけの人は、この章は読まなくてOKです）

### 大事な前提

服のデータは「ブラウザごと」に保存されます。あなたが追加した服は、そのままでは他の人には見えません。全員に配るには、下の手順で「書き出して → 反映」が必要です。

### 手順

1. 公開URLで、いつものパソコンで服を追加・整理する
2. 画面上部の「全部まとめて書き出す（管理者用）」ボタンを押す
   → `服データ全体_日付.json` というファイルがダウンロードされます（今ある全部が入ります）
3. そのファイルを管理担当（Claude）に渡して「反映して」と伝える
   → 追加した服も、削除した服も、全員のツールに同期されます

### 注意

- 作業するパソコン・URLは1つに固定してください（あちこちで追加するとバラけます）
- 「この服は全員から消したい」場合も、その状態で書き出せば反映されます

## 困ったときは（トラブルシューティング）

### 貼り付け（Cmd+V / Ctrl+V）で画像が入らない

- 画像を「画像をコピー」でコピーできているか確認してください
- うまくいかないときは、画像をいったんパソコンに保存して、ドラッグ＆ドロップで追加してみてください

### 追加した服が消えた／戻ってきた

- 服データはブラウザごとの保存です。別のパソコンや別のブラウザで開くと、内容が違って見えます
- いつも同じパソコン・同じURLで開くようにしてください

### 更新したのに反映されない

- 公開URLは、更新直後は少しの間、古い状態が表示されることがあります。数分待ってから、ページを再読み込み（リロード）してみてください

### ChatGPTで「コンテンツポリシー違反の可能性」と出る

- 多くの場合は誤判定です。同じプロンプトでもう一度送ると通ることがあります
- それでもダメなときは、管理担当に伝えてもらえれば、文言を調整します

## よくある質問

**Q. 私が追加した服は、他の人にも見えますか？**
A. そのままでは見えません。全員に配るには、管理者向けの手順（全部まとめて書き出す → 反映）が必要です。

**Q. 服やモデルはこのあとも増えますか？**
A. はい、随時追加していきます。「こんな服も入れてほしい」があれば教えてください。

**Q. スマホでも使えますか？**
A. 使えます。ただし服を貼り付けたり書き出したりする作業は、パソコンのほうがやりやすいです。

**Q. バッグの持ち方や服の丈がうまく再現されないことがあります**
A. AI画像生成が苦手な細かい部分です。うまくいかないときは、できた画像に「バッグを○○の持ち方にして」などと追加で指示すると直しやすいです。
